# Vistara Learn

## Overview

Vistara Learn is a web-based learning and career development platform designed to support students and professionals in achieving their academic and career goals. The platform combines education, career guidance, visa consultation, and skill development services in a single user-friendly interface.

## Features

* Career and Job Opportunities Portal
* Student and Work Visa Consultation Services
* IELTS, TOEFL, and GRE Preparation Courses
* Mock Test and Assessment System
* Video-Based Learning Portal
* Student Registration and Authentication
* Admin Dashboard for Content Management
* Responsive and Modern User Interface
* Dynamic Background and Interactive Components

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Font Awesome
